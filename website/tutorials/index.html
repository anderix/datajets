DataJets Tutorials

So you've decided you don't want to waste any more time writing boring JDBC and SQL code to do the same thing over and over again. You've downloaded the DataJets library and added it to your classpath. Now you're ready to get started! Well, you're in the right place! These tutorials will show you how to use the DataJets framework in your projects. If you haven't yet downloaded the library, please do so now and then come back here to get started with Hello, DataJet!.

To use these tutorials:

The tutorials on this site assume you'll be writing your code on a Windows-based computer, but I'll keep everything generic enough so it will be useful to you no matter what your development machine is. You'll need to have a database and an appropriate JDBC driver to execute the code. If you need a refresher, check out Establishing a Connection. That's pretty much all the JDBC you'll need to know to use DataJets. If you're working on a team and someone has already created a ConnectionGenerator for you, you won't even need to know that!

For examples, I'll be using a Microsoft Access database with an ODBC connection and the JDBC-ODBC Bridge driver. I'll supply a downloadable copy of the database for each tutorial, but you'll have to set up the ODBC data source yourself. The tutorials assume a level of familiarity with Access, but if you would rather run the code examples using another database you shouldn't have a problem following along. Just remember to change your database driver and connection string. You won't need any specific development environment; I'll be using javac and a basic text editor.
