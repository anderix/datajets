Why did you create DataJets?
I got tired of writing the same SQL code over and over again for applications that were only slightly different, and I wanted classes that would handle all the mechanics of CRUD database functionality for me. I wasn't completely happy with any of the other libraries that exist today, so I wrote my own.

How much does DataJets cost?
DataJets is free software and a free download. There is no cost to use the library or source code.

What are the licensing terms?
DataJets is open source software released under the LGPL.

I'm a developer. What does DataJets offer me?
First and foremost, DataJets offers you freedom from mundane SQL tasks. It allows you to focus on designing business objects for your applications instead of the mechanics of database functions. DataJets is object oriented and open source, so you benefit from other people's efforts by reusing their code. DataJets has a low learning curve, so you'll be more productive almost immediately.

I'm a business user. What does DataJets offer me?
DataJets allows developers to focus more on your needs when developing software. You don't care about how your data is stored in a database. By using DataJets, developers won't have to think about this either.
Return to top of page
I'm a project manager. What does DataJets offer me?
Using DataJets on your project will decrease your development cycles for many solutions. The library has a very low learning curve, and developers on your project will be able to build applications faster. Since they will be using code that has already been tested, you'll spend less time in the testing phase. This translates to faster time-to-market for your applications, with lower development costs.

I know SQL very well. Why should I use DataJets?
Great! I think everyone should know SQL, and know it well! DataJets will let you leverage your SQL skills. The power of the SQL language is always available to you when coding, and knowledge of SQL will let you accomplish more with your DataJets. Your own SQL statements are supported, and the SqlBuilder will ease this task.

How do I use DataJets?
There are many ways to use DataJets. You can pick and choose which parts of the framework you would like to use. This site has several tutorials and extensive documentation to help you get started. The learning curve for basic DataJets is low. A good place to start is Hello, DataJet!
Return to top of page
How does DataJets compare to other similar libraries?
The concept behind DataJets is not new. There are many other alternatives that accomplish the same or similar things. DataJets is much easier to use than any other framework I've seen. It is especially well suited to Web-based applications that must provide ways for users to maintain data.

Can I use DataJets with stored procedures?
No. DataJets builds SQL statements dynamically within code. Stored procedures are not flexible enough to handle these interactions.

I've got an idea for a new feature. Can DataJets do XYZ?
Yes! DataJets is open source, so if you have an idea you are free to implement it. Under the terms of the LGPL, you must release any extensions or changes to the library back to the open source community. However, products the use or link to the DataJets library may be closed source and sold as commercial software.
